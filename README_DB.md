# מערכת ניהול שירים ותוכן למשרד

## מבוא
מערכת מאוחדת לניהול שירים, לוגואים, פלאגינים ותוכן לרשתות חברתיות.
המערכת משלבת את שתי המערכות הקיימות למערכת אחת מקיפה עם עיצוב אחיד.

## התקנה

### 1. הגדרת מסד נתונים

הרץ את קובץ הסכמה ליצירת מסד הנתונים והטבלאות:

```bash
mysql -u adm_tkc -p'a1367$1661va' < database/schema.sql
```

או דרך phpMyAdmin:
1. התחבר ל-phpMyAdmin
2. צור מסד נתונים חדש בשם `songs_management`
3. בחר במסד שנוצר
4. לחץ על "Import" ובחר את הקובץ `database/schema.sql`

### 2. הרשאות תיקיות

ודא שהתיקיות הבאות קיימות וניתנות לכתיבה:

```bash
chmod 755 public/uploads/songs
chmod 755 public/uploads/logos
chmod 755 public/uploads/plugins
chmod 755 public/uploads/images
chmod 755 public/uploads/story_content
chmod 755 songs/uploads
```

### 3. יצירת משתמש ראשון

לאחר הרצת הסכמה, המערכת תכלול משתמש admin בסיסי.
ניתן ליצור משתמש נוסף דרך: `/public/create_admin.php`

## מבנה מסד הנתונים

המערכת כוללת את הטבלאות הבאות:

### ניהול משתמשים
- `roles` - תפקידים
- `users` - משתמשים
- `user_permissions` - הרשאות משתמשים

### ניהול תוכן
- `categories` - קטגוריות
- `songs` - שירים (מאוחד משתי המערכות)
- `song_tags` - תגיות לשירים
- `song_tag_relations` - קישורי שירים לתגיות
- `song_images` - תמונות/PNG של מילים לשיר
- `song_metadata` - מטא-דאטה נוספת לשירים

### ניהול לוגואים ופלאגינים
- `logos` - לוגואים למפיקים ואירועים
- `plugins` - פלאגינים לתוכנות עריכה
- `story_content` - תוכן לסטוריז (מוזיקה + לוגואים)

### ניהול רשתות חברתיות
- `clients` - לקוחות
- `post_platforms` - פלטפורמות (פייסבוק, אינסטגרם וכו')
- `social_posts` - פוסטים לפרסום

### מעקב ופעילות
- `activity_log` - לוג פעילות במערכת

## תכונות עיקריות

### ניהול שירים
- ✅ העלאת שירים (MP3, WAV)
- ✅ קישורים לגוגל דרייב
- ✅ קישורי יוטיוב
- ✅ העלאת PNG של מילים (מספר עמודים)
- ✅ יצירת ZIP אוטומטי של כל תמונות המילים
- ✅ מטא-דאטה מלאה (אמן, אלבום, משך זמן)
- ✅ קטגוריות ותגיות

### ניהול לוגואים
- ✅ העלאת לוגואים (PNG, SVG)
- ✅ שמות בעברית ואנגלית
- ✅ סיווג לפי סוג (מפיק, מותג, אירוע)
- ✅ חיפוש מהיר

### ניהול פלאגינים
- ✅ העלאת פלאגינים
- ✅ סיווג לפי סוג (audio, video, graphics)
- ✅ רשימת תוכנות תואמות
- ✅ ניהול גרסאות

### תוכן לסטוריז
- ✅ שילוב מוזיקה + לוגואים
- ✅ תבניות מוכנות
- ✅ ניהול משך זמן
- ✅ תגיות לארגון

### רשתות חברתיות
- ✅ ניהול פוסטים למספר פלטפורמות
- ✅ תזמון פרסום
- ✅ מעקב סטטוס (טיוטה, מתוזמן, פורסם)
- ✅ קישור ללקוחות

## עיצוב המערכת
